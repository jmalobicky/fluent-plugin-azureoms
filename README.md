## Installation

### RubyGems

```
$ gem install fluent-plugin-azureoms
```

### Bundler

Add following line to your Gemfile:

```ruby
gem "fluent-plugin-azureoms"
```

And then execute:

```
$ bundle
```

You can generate configuration template:

```
$ fluent-plugin-config-format output azureoms
```
